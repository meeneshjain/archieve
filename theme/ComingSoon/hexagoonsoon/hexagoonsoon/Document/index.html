I) HTML Structure - top

Hexagoon is a fluid theme with indented and intuitively spaced HTML. You will find in-line commenting allthroughout the HTML so that you can always figure out what section of the layout you are in. The backbone of every page is as follows:

<div id="md-comingsoon" class="md-comingsoon">
    <div class="md-mask"></div>
        <div id="ri-grid" class="ri-grid ri-grid-size-2 ri-shadow clearfix">
        	{Animated Responsive Image Grid}
		</div>
	</div><!-- #ri-grid -->
    <div class="md-wrap">
    	<div id="md-sections" class="md-sections">
        	{Section code}
        </div><!-- /.md-sections -->
        
        <div id="md-arrow" class="md-arrow">
            <div id="arrow-prev" class="icon-angle-up md-arrow-top"></div>
            <div id="arrow-next" class="icon-angle-down md-arrow-bottom"></div>
        </div><!-- /.md-arrow -->
        
        <div class="md-pages clearfix">
            {Paging code}
        </div><!-- /.md-pages -->
	</div><!-- /.md-wrap -->
</div>
Style options

You can change style (green with white opacity) in a second by chhange body class name from .md-option-1 to .md-option-2

You can add new style by copy CSS code from style.css and create your own.

CountDown Section

CountDown Section is simple, it contains logo and count down numbers. Here is the HTML structure

			<section class="md-section md-home">
            	<div class="md-content">
                	<div class="md-logo">
                    	<h1 class="rs"><a href="#"></a></h1>
                        </div>
                        <h1 class="md-title">Our website is coming soon</h1>
                        <div id="defaultCountdown" class="clearfix"></div><!-- /countdown -->
            	</div><!-- /.md-content -->
            </section><!-- /.md-home -->

Notify form

Notify form auto save emails to server/emails.txt file. You can change file name/location by edit server/notify.php file, line 40:

 
$file = fopen('emails.txt', 'a+');
Contact form

To configure contact form send email to your email address, open server/message.php, find line 47:

$to = "info@megadrupal.com";

and change email address to your email address

Domain Sale form

To configure domain sale form send email to your email address, open server/domain.php, find line 56:

$to = "info@megadrupal.com";

and change email address to your email address

Image Grid Background

To use image grid background, just insert this HTML code before <div class="md-wrap"> code

 
<div class="md-mask"></div>
	<div id="ri-grid" class="ri-grid ri-grid-size-2 ri-shadow clearfix">
    	<ul class="rs">
        	<li><a href="#"><img src="img/items/item1.jpg" alt=""/></a></li>
            {YOUR IMAGE HERE}
		</ul>
	</div><!-- #ri-grid -->
Fullscreen Image Slideshow

To use fullscreen image slideshow, open script.js, go to line 127 and edit to your image url

 
{image : 'img/s1.jpg'},
{image : 'img/s2.jpg'},
{image : 'img/s3.jpg'},
{image : 'img/s4.jpg'},
{image : 'img/s5.jpg'}

And add this HTML code before close body tag

 <div id="supersized-loader"></div><ul id="supersized"></ul>
Fullscreen Image Section Background

To use fullscreen image section background, just insert this code before close section tags:

 
<div class="md-mask"></div>
<div class="md-bg-image"><img src="img/s2.jpg" alt=""></div>
Fullscreen Video Background

To use fullscreen video background, just insert this HTML code before <div class="md-wrap"> code

 
<div class="ri-grid ri-grid-size-2 ri-shadow clearfix">
	<div class="ri-blue"></div>
    <video width="1920" height="1080" autoplay loop>
    	<source src="video/video.mp4" type="video/mp4">
        <source src="video/video.ogg" type="video/ogg">
        <source src="video/video.ogg" type="video/ogg">
        Your browser does not support the video tag.
	</video>
</div><!-- #ri-grid -->

II) CSS Files and Structure - top

This theme uses style.css file as main CSS file.

style.css

Main CSS files in this theme is style.css that contains all of specific stylings for the theme.The file is separated into 10 sections using labels as follows:

		
1.	Reset
2.	Common style
3.	Button style
4.	Options style (you can add your style here)
5.	Home style
6.	Notify style
7.	Contact style
8.	Domain for sale style
9.	Follow style
10.	Animated Responsive Image Grid

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

responsive.css

responsive.css contains styling for media queries that allows us to tailor to different resolutions and devices without changing the content. The file is separated into sections as follows:

	
@media screen and (max-width: 1024px) {
	
{SOME CODES}

}

@media screen and (max-width: 700px ) {
	
{SOME CODES}

}

@media screen and (max-width: 570px) {

{SOME CODES}

}

@media screen and (max-width: 380px) {

{SOME CODES}

}

@media screen and (max-width: 280px) {

{SOME CODES}

}

If you would like to edit styling of the specific device width, simply find the appropriate label in the responsive.css file, and then scroll down until you find the appropriate style that needs to be edited.

III) JavaScript - top

This theme imports following Javascript files.

  1. jquery-1.9.1.min.js - jQuery library
  2. jquery.countdown.js - jQuery Count down
  3. jquery.cycle2.js - jQuery cycle 2
  4. jquery.easing.min.js - Easing library
  5. jquery.form.js - allows you to upgrade HTML forms to use AJAX
  6. jquery.gridrotator.js - Animated Responsive Image Grid
  7. jquery.mousewheel.js - Mousewheel library
  8. jquery.tweet.min.js - Get tweet from Twitter
  9. jquery.ulslide.js - Section slide
  10. jquery.validate.min.js - validate contact form and comment form
  11. .js -
  12. modernizr.js - Modernizr library
  13. supersized.3.2.7.js - Fullscreen background slideshow
  14. supersized.shutter.js - Fullscreen background slideshow theme
  15. script.js - contains all custom script

Except script.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.

script.js contains custom scripts and scripts to call method for above plugin. The file is separated in to sections with appropriate label.

To configure Twitter, you need to change script.js file at line 66 and 69. You need to have a twitter App for your usage in order to obtain OAuth credentials, see https://dev.twitter.com/apps for help.

To edit count down time, you can change it by edit script.js file, line 46: austDay = new Date(2014, 9, 26);
first number is year, second is month (from 0 to 11), third is day, you can add hour & minute too (by adding more number after 26).
